Como jugar:



Explicación: El jugador y la CPU acuerdan un numero total por el que empezar a restar, luego, acuerdan el maximo numero por el que se va a poder restar. El primero que logre
 llegar a 0 es el ganador. La trampa de este juego es que es imposible ganarle a la CPU hagas lo que hagas.



1- Elige cualquier numero para que sea este el total desde el que empezar a restar.



2- Elige el maximo permitido para restar (se podra restar 1 hasta el maximo acordado).



3- empieza a restar y trata de ganarle a la CPU.
